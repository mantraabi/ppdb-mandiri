**PPDB Mandiri** adalah platform digital berbasis web yang dirancang khusus untuk memodernisasi dan menyederhanakan proses pendaftaran siswa baru di berbagai jenjang sekolah.

## ✨ Fitur Utama

- **Dashboard Admin Pro:** Ringkasan statistik real-time dengan grafik interaktif (ApexCharts).
- **Manajemen Pendaftar:** Filter data pendaftar, verifikasi berkas, dan ekspor data ke Excel.
- **Panel CPD (Siswa):** Alur pendaftaran mandiri mulai dari biodata, unggah dokumen, hingga cetak bukti pendaftaran.
- **Notifikasi Real-time:** Integrasi Socket.io untuk pemberitahuan instan saat ada pendaftar baru.
- **Responsive Design:** Optimal diakses melalui perangkat mobile maupun desktop menggunakan Tailwind CSS.
- **Pengaturan Instansi:** Admin dapat menyesuaikan logo, kop surat, dan informasi sekolah secara dinamis.

## 🛠️ Tech Stack

- **Framework:** [Vue.js 3](https://vuejs.org/) (Composition API)
- **Build Tool:** [Vite](https://vitejs.dev/)
- **Styling:** [Tailwind CSS](https://tailwindcss.com/)
- **Icons:** [Lucide Vue Next](https://lucide.dev/)
- **Charts:** [ApexCharts](https://apexcharts.com/)
- **State Management:** [Pinia](https://pinia.vuejs.org/)
- **HTTP Client:** [Axios](https://axios-http.com/)
